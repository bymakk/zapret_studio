# zapret-cb-sc

Форк [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube) для **Windows**: обход DPI с готовыми списками хостов под кам-сайты (Chaturbate, Stripchat, MyWebcamRoom, Camsoda и связанные CDN/API), плюс правки запуска и проверки `winws`.

> **Важно.** Инструмент меняет сетевой трафик на вашей машине. Запускайте только от **администратора**, из доверенного архива, на свой страх и риск.

## Чем отличается от исходника

- В **`lists/list-general-user.txt`** добавлены домены для перечисленных сайтов и их инфраструктуры (по данным [urlscan.io](https://docs.urlscan.io/guides/quickstart) и ручной доработке).
- Файл **`lists/list-ipset-host-skip.txt`**: для хостов Camsoda отключено второе срабатывание правил по **ipset** (частая причина `ERR_CONNECTION_RESET` при работающем остальном трафике).
- Перед стартом **`general*.bat`** вызывают **`service.bat prepare_winws`**: останавливаются предыдущий `winws.exe`, сервис `zapret` (если был) и очищается WinDivert — можно переключать пресеты без ручной чистки.
- После запуска вызывается **`verify_winws`**: проверка, что процесс `winws.exe` поднялся, и контроль TCP timestamps.

## Требования

- **Windows** (x64), права **администратора**.
- Каталог без «кривых» путей: лучше распаковать в корень диска, например `C:\zapret-cb-sc` (пробелы и длинные пути иногда мешают сервису и батникам).

## Быстрый старт

1. Нажмите **Code → Download ZIP** и распакуйте архив.
2. Откройте папку с **`bin`**, **`lists`**, **`service.bat`**.
3. Запустите нужную стратегию **от имени администратора**, например:
   - `general (ALT).bat` — один из основных вариантов;
   - при необходимости попробуйте `general (ALT2).bat`, `general.bat` и другие `general*.bat` (разные параметры DPI).
4. В консоли должны появиться сообщения проверки; при ошибке запуска скрипт остановится и подскажет, что проверить.

### DNS

Если отдельные сервисы (в т.ч. YouTube) режутся по **DNS**, в браузере или в системе имеет смысл включить **Secure DNS** (например, Google DNS / другой доверенный провайдер) — это не часть самого zapret, но часто нужно для полноценного доступа.

## Остановка обхода

**Вариант А — только процесс:**

```bat
taskkill /IM winws.exe /F
```

**Вариант Б — сервис + процесс + WinDivert** (как в меню `service.bat`):

```bat
net stop zapret
sc delete zapret
taskkill /IM winws.exe /F
net stop WinDivert
sc delete WinDivert
```

Или запустите **`service.bat`** и выберите пункт **Remove Services**.

## Настройка списков

| Файл | Назначение |
|------|------------|
| `lists/list-general-user.txt` | Ваши домены поверх общего списка Discord/и т.д. |
| `lists/list-exclude-user.txt` | Исключения из hostlist-правил |
| `lists/list-ipset-host-skip.txt` | Хосты, для которых не применяется «второй» слой правил по ipset (см. Camsoda) |
| `lists/ipset-exclude-user.txt` | Исключения по IP для ipset-правил |

Пустые шаблоны для части файлов создаёт **`service.bat`** при вызове `load_user_lists` (при первом запуске батников).

## Сервис Windows

Установка/удаление службы — через **`service.bat`** (меню). После установки службы отдельные `general*.bat` для повседневного запуска обычно не нужны: стратегия крутится как сервис.

## Лицензия и ответственность

Состав списков и параметры обхода могут устаревать при смене CDN и политик сайтов. Авторы репозитория не гарантируют работоспособность на всех провайдерах; соблюдайте законы страны пребывания и правила сети, к которой подключены.
